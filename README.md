Description:

This is my personal portfolio website, built using HTML and CSS, to showcase my projects, experiences, and skills in a clean, organized way. The website is divided into multiple sections that highlight my background, technical abilities, and achievements.

Features:

- Responsive Design: Works on desktop, tablet, and mobile devices.

- Sectioned Layout: Includes sections such as:

1. Home / About Me – Introduction and background information

2. Skills – Technical and soft skills overview

3. Projects – Highlighted personal and academic projects

4. Experience / Activities – Work experience, volunteering, and leadership roles

5. Contact – Easy way to get in touch

   
Technologies Used

- HTML

- CSS
